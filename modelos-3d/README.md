# Modelos 3D

Esta pasta contém arquivos relacionados à estrutura mecânica do braço robótico.

## Origem dos modelos

Os modelos 3D utilizados neste projeto foram obtidos a partir do projeto:

**ESP32 BT Robotic Arm with High Torque Servos + Android App**  
Criado por: DavTech_3D  
Disponível em: https://www.thingiverse.com/thing:6896222  
Licença: Creative Commons - Attribution - Share Alike

## Observação

Os modelos foram utilizados como base para a montagem do protótipo físico do TCC. 
Algumas peças podem ter sido adaptadas conforme as necessidades do projeto, como ajustes de montagem, encaixe dos servos motores e organização estrutural.
